# Calculadora-1DAM-Contornos-CIFPRU
Proyecto colaborativo que implementa una calculadora en Java para el aula de 1º de DAM en el módulo de Contornos de Desenvolvemento en el centro CIFP Rodolfo Ucha, Ferrol.

## Operaciones implementadas:

 - Suma: a + b. Nombre de operación: "add"
 - Resta: a - b. Nombre de operación: "subtract"
